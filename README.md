# SRT Voice App

Vibecode з Claude - Застосунок для озвучування українських субтитрів з автоматичною синхронізацією таймінгу.

## Можливості

- Озвучка SRT субтитрів українською мовою
- Підтримка 4 TTS движків:
  - **Edge TTS** (онлайн, Microsoft) - 2 голоси
  - **Piper TTS** (офлайн) - 3 голоси
  - **MMS TTS** (офлайн, Meta) - 1 голос
  - **UA-ESPNET** (онлайн) - 5 голосів
- Автоматична синхронізація з таймінгом субтитрів
- Додавання тиші між субтитрами, на початку та в кінці
- Обробка порожніх субтитрів як тиші
- Пакетна обробка кількох файлів з індивідуальною тривалістю
- Підгонка під точну тривалість відео
- Передпрослуховування голосів

## Встановлення

### 1. Python та залежності
```bash
# Встановіть Python 3.10+
pip install -r requirements.txt
```

### 2. FFmpeg

- Завантажте з https://www.gyan.dev/ffmpeg/builds/
- Розпакуйте та додайте `bin/` папку до PATH

### 3. Piper TTS (опціонально, для офлайн роботи)

1. Завантажте Piper з https://github.com/rhasspy/piper/releases
2. Помістіть `piper.exe` в папку `piper/`
3. Завантажте українські моделі:
```bash
pip install huggingface-hub
python -c "from huggingface_hub import hf_hub_download; hf_hub_download(repo_id='rhasspy/piper-voices', filename='uk/uk_UA/ukrainian_tts/medium/uk_UA-ukrainian_tts-medium.onnx', cache_dir='piper_voices'); hf_hub_download(repo_id='rhasspy/piper-voices', filename='uk/uk_UA/ukrainian_tts/medium/uk_UA-ukrainian_tts-medium.onnx.json', cache_dir='piper_voices')"
```

## Використання

1. Запустіть програму:
```bash
python main.py
```

2. Оберіть SRT файл(и)
3. Виберіть движок TTS та голос
4. (Опціонально) Вкажіть тривалість відео для точної синхронізації
5. Натисніть "Запустити озвучку"
6. Дочекайтесь завершення - MP3 файл з'явиться в тій же папці

## Особливості

- **Синхронізація з таймінгом**: Кожен субтитр озвучується та підганяється під час в SRT
- **Тиша**: Автоматично додається тиша між субтитрами, на початку та в кінці
- **Порожні субтитри**: Обробляються як тиша відповідної тривалості
- **Пакетна обробка**: Можна обрати кілька файлів та вказати тривалість для кожного
- **Зупинка**: Можна зупинити обробку в будь-який момент

## Вимоги

- Python 3.10+
- FFmpeg
- Інтернет (для Edge TTS та UA-ESPNET)
- ~500 МБ для MMS моделі (завантажується автоматично)
- ~77 МБ для Piper моделі (опціонально)
